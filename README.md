Мобильное Android-приложение для планирования тренировок, отслеживания прогресса и получения рекомендаций по тренировочному процессу.
Проект разработан в рамках дипломной работы и как pet-project для прокачки навыков Android-разработки.

Функциональность:

- Просмотр тренировочных программ
- Каталог упражнений по группам мышц
- Видео с техникой выполнения упражнений
- Календарь тренировок
  - добавление тренировок по датам
  - фиксация веса, подходов и повторений
- Экран прогресса
  - график изменения веса по упражнениям
- Рекомендации по питанию
- Удобная навигация между разделами

Архитектура

Проект построен с использованием современных подходов Android-разработки:

- MVVM
- Разделение на `UI / ViewModel / Data`
- Работа с локальной базой данных

Технологии и инструменты

Основной стек
- Kotlin
- Jetpack Compose
- Material 3
- Navigation Compose

Хранение данных
- Room
- DAO
- Entities
